## 🧭 Overview

This repository documents my structured **15-day SOC Analyst Linux training** using **Linux Mint**. The goal is to build real-world, job-ready skills required for entry-level **SOC (Security Operations Center)** roles through hands-on, scenario-based practice.

### Training Focus

| Area | Description |
|---|---|
| 🖥️ Linux Operations | Core command-line navigation and system control |
| 📁 Evidence Handling | File operations relevant to investigations |
| 📊 Log Analysis | Reading and interpreting system/security logs |
| 🔍 Investigation | Process and file-level threat hunting |
| 🛡️ Blue Team Mindset | Thinking like a defender |

## 🎯 Learning Objectives

By the end of this training, I will be able to:

- [x] Navigate Linux systems efficiently
- [x] Perform file operations (create, copy, move, delete, edit)
- [x] Analyze system and security logs
- [ ] Identify suspicious processes and activity
- [ ] Understand file permissions and user behavior
- [ ] Perform basic incident investigation tasks
- [ ] Prepare for SIEM tools like Splunk and Wazuh
